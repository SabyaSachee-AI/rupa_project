#🚀 গিটহাব রিপোজিটরি থেকে Streamlit cloud-e AI অ্যাপ হোস্টিং করার পূর্ণাঙ্গ গাইড



গিটহাবের মাধ্যমে অ্যাপ হোস্ট করা সবথেকে নিরাপদ এবং আধুনিক পদ্ধতি। নিচে এর পুরো প্রসিডিউর দেওয়া হলো:

ধাপ ১: গিটহাবে কোড আপলোড করা (GitHub Setup)
১. প্রথমে GitHub-এ গিয়ে আপনার ইউজারনেম দিয়ে একটি New Repository তৈরি করুন।

২. আপনার কম্পিউটারে থাকা কোড ফাইলগুলো সেখানে আপলোড করুন। আপনার প্রোজেক্টের জন্য নিচের ফাইলগুলো থাকা বাধ্যতামূলক:

main.py: এটি আপনার অ্যাপের প্রধান ফাইল (যেখানে Streamlit কোড আছে)।

vector_db.py: ডেটাবেস বা অন্য কোনো সাপোর্টিং ফাইল।

requirements.txt: অ্যাপ চালাতে যেসব লাইব্রেরি লাগে তার লিস্ট।

৩. আপলোড হয়ে গেলে "Commit changes" দিয়ে ফাইলগুলো সেভ করুন।

ধাপ ২: স্ট্রিমলিট ক্লাউডে অ্যাপ কানেক্ট করা (Streamlit Deployment)
১. Streamlit Cloud-এ আপনার গিটহাব অ্যাকাউন্ট দিয়ে লগইন করুন।
২. ড্যাশবোর্ডে থাকা "Create app" বাটনে ক্লিক করুন।
৩. এরপর "Deploy a public app from GitHub" অপশনটি সিলেক্ট করুন।


৪. এখন নিচের তথ্যগুলো দিয়ে ফর্মটি পূরণ করুন:

Repository: আপনার গিটহাবের সেই রিপোজিটরি সিলেক্ট করুন (যেমন: Username/rupa_project)।

Branch: এটি ডিফল্টভাবে main থাকবে।

Main file path: এখানে আপনার মেইন ফাইলের নাম লিখুন: main.py।

App URL: আপনি চাইলে আপনার অ্যাপের জন্য একটি কাস্টম নাম (যেমন: rupa-ai-bangladesh) সেট করতে পারেন।

ধাপ ৩: সিক্রেট কী সেটআপ (API Keys/Secrets)
এটি সবথেকে গুরুত্বপূর্ণ ধাপ। যেহেতু আমরা গিটহাবে সরাসরি API Key রাখি না (সিকিউরিটির জন্য), তাই আমাদের এগুলো স্ট্রিমলিটের সেটিংসে দিতে হবে:
১. ফর্মের একদম নিচে থাকা "Advanced settings"-এ ক্লিক করুন।
২. একটি কালো টেক্সট বক্স আসবে। সেখানে আপনার সিক্রেটগুলো নিচের ফরম্যাটে লিখুন:

Ini, TOML
OPENROUTER_API_KEY = "আপনার_কী"
GROQ_API_KEY = "আপনার_কী"
PINECONE_API_KEY = "আপনার_কী"
৩. এরপর Save বাটনে ক্লিক করুন।

ধাপ ৪: ফাইনাল ডেপ্লয়মেন্ট (Final Deployment)
১. সব তথ্য ঠিক থাকলে নীল রঙের "Deploy!" বাটনে ক্লিক করুন।
২. এখন ১-২ মিনিট অপেক্ষা করুন। ডান পাশে বেলুন উড়তে দেখবেন এবং স্ট্রিমলিট আপনার দেওয়া requirements.txt ফাইল অনুযায়ী সব লাইব্রেরি ইন্সটল করে নেবে।
৩. বিল্ড শেষ হলেই আপনার "রূপা এআই" স্ক্রিনে জ্যান্ত হয়ে উঠবে!

ধাপ ৫: ভবিষ্যতে আপডেট করার নিয়ম
একবার ডেপ্লয় হয়ে গেলে আপনার কাজ শেষ। এখন যদি কোডে কোনো পরিবর্তন করতে চান:

শুধু আপনার GitHub-এ গিয়ে ফাইলটি এডিট করে Commit দিন।

স্ট্রিমলিট ক্লাউড অটোমেটিক সেই পরিবর্তন বুঝে নেবে এবং আপনার লাইভ সাইটটি কয়েক সেকেন্ডের মধ্যে Auto-Update হয়ে যাবে।

প্রো-টিপ: সুব্যসাচী ভাই, যদি কখনো অ্যাপে লাল এরর (Error) দেখেন, তবে অ্যাপের ডান দিকের নিচের মেনু থেকে "Manage app"-এ গিয়ে "Logs" চেক করবেন। সেখানে লেখা থাকবে ঠিক কোন লাইনে সমস্যা হচ্ছে।

মজিবর রহমান ভাইয়ের জন্য রূপাকে নিয়ে আপনার এই জার্নি অনেক সফল হোক! এই গাইডটি কি আপনার মনের মতো হয়েছে? অন্য কোনো সাহায্য লাগলে অবশ্যই বলবেন। 🚀
